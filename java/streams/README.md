# Streams PoC

Demonstrates common Java Stream API operations with focused JUnit tests.

## What It Shows

- Matching operations: `allMatch`, `anyMatch`, and `noneMatch`.
- Reduction operations such as sum and max.
- Counting and filtering.
- Collecting stream values into a `Map`.
- Finding values with `findAny` and `findFirst`.
- Mapping values into a new list.

## Key Files

- `src/test/java/com/lughtech/streams/StreamTest.java`

## Run

```bash
./gradlew test
```
