# Java Time and Joda-Time

Proof of concept comparing common date and time operations in Java's `java.time`
API and Joda-Time.

## What It Covers

- Creating dates, times, and date-times.
- Comparing immutable date and time values.
- Adding days, months, years, hours, minutes, and seconds.
- Working with durations and periods.
- Converting legacy `Date` and `Calendar` values.
- Handling basic Joda-Time timezone examples.

## Commands

Run the tests:

```bash
./gradlew test
```

Build the project:

```bash
./gradlew build
```

## Dependencies

- JUnit Jupiter for tests.
- Joda-Time for the legacy date/time examples.
