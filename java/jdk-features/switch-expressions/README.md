# Switch Expressions

Proof of concept for Java switch expressions.

## What It Covers

- Parsing a simple arithmetic expression.
- Selecting arithmetic behavior with switch expression arrow labels.
- Returning values directly from switch branches.
- Testing addition, subtraction, multiplication, and division cases.

## Commands

Run the tests:

```bash
./gradlew test
```

Build the project:

```bash
./gradlew build
```
